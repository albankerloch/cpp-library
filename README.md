# C++ library

For pedagogical purposes, I recoded 4 C++ containers :
- map
- stack
- vector
- list

Features:
- Same behavior as the C++ containers
- Testing files
- No memory leak

## Installation

Use this command in a terminal : 

```bash
make
```

(make and clang++ needed)

## Usage

- Add #include "Map.hpp" in a cpp file (like in Test_Map.cpp)
